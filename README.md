# <div align="center">**✨VALORANT COMPANION✨**
</div>

#### <div align="center">***- Best Valorant Companion for Beginners 👀***
</div>

Valorant Companion✨ is an Android application designed and developed to provide players🎮 with comprehensive details on agents, maps, game modes, and weapons. This application is developed using Modern UI-toolkit Jetpack Compose🚀(Kotlin) for android. It makes use of various different libraries📚 and dependencies like Material UI 3, Retrofit, Coil, ReadMoreTextView, Dagger-Hilt, Datastore Preferences, etc. It also has dark and light themes for increasing user experience and overall engagement.

### **// Features⚡**

- **Agent Profiles**: Detailed information about each agent, including abilities, roles, and tips for optimal use.
- **Map Insights**: In-depth breakdowns of maps with key locations, strategies, and navigation tips.
- **Game Modes**: Overview and strategies for each game mode to help you adapt and excel.
- **Weapon Stats**: Detailed stats and comparisons for all weapons in the game.

---

### **// MVVM Clean Architecture 🪨**
- **Data**: Data sources and operations that provide the data to the app.
- **Model**: Data classes and entities that represent the application's data.
- **Domain**: Business logic and data operations that manage the data and UI state.
- **Repository**: Data sources and operations that provide the data to the ViewModel.
- **UseCase**: The use cases that define the operations that can be performed on the data.
- **Di**: Dependency injection modules and components that provide dependencies to the app like API, ViewModel, Repository, etc.
- **ViewModel**: Business logic and data operations that manage the data and UI state.
- **View**: UI components and screens that display the data and interact with the user.
- **Utils**: It contains helper classes and functions that are used throughout the app.

---

### **// Libraries and Dependencies 🤖**
  
- **Jetpack Compose**: Modern declarative UI toolkit for building native UI.
- **Material UI 3**: Material Design components for Jetpack Compose.
- **Dagger Hilt**: Dependency injection library for Android.
- **KSP**: Kotlin Symbol Processing for Dagger Hilt.
- **Retrofit**: Type-safe HTTP client to make API calls.
- **Coil**: Image loading library to fetch and display images.
- **Datastore**: Datastore Preferences for storing local key-value pairs.
- **ReadMoreTextView**: Custom TextView for expanding and collapsing text.

  cd ValorantCompanion

- Run the project on an emulator or physical device.
